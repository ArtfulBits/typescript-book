# Статичні конструктори в TypeScript

Класи TypeScript (як і класи JavaScript) не можуть мати статичного конструктора. Однак, ви можете добитися того ж ефекту досить легко, просто викликавши його самостійно:

```ts
class MyClass {
    static initialize() {
        // Ініціалізація
    }
}
MyClass.initialize();
```

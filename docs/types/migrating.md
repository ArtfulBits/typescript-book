### Suppressing Errors
TypeScript –Ω–µ–≥–∞–π–Ω–æ –ø–æ—á–Ω–µ –ø–µ—Ä–µ–≤—ñ—Ä–∫—É —Ç–∏–ø—É –≤–∞—à–æ–≥–æ –∫–æ–¥—É, —ñ –≤–∞—à –≤–∏—Ö—ñ–¥–Ω–∏–π –∫–æ–¥ JavaScript *–º–æ–∂–µ –±—É—Ç–∏ –Ω–µ —Ç–∞–∫–∏–º –æ—Ö–∞–π–Ω–∏–º, —è–∫ –≤–∏ –¥—É–º–∞–ª–∏*, —ñ, –æ—Ç–∂–µ, –≤–∏ –æ—Ç—Ä–∏–º—É—î—Ç–µ –¥—ñ–∞–≥–Ω–æ—Å—Ç–∏—á–Ω—ñ –ø–æ–º–∏–ª–∫–∏. –ë–∞–≥–∞—Ç–æ –∑ —Ü–∏—Ö –ø–æ–º–∏–ª–æ–∫ –º–æ–∂–Ω–∞ –ø—Ä–∏–¥—É—à–∏—Ç–∏ –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é `any`, –Ω–∞–ø—Ä–∏–∫–ª–∞–¥:

```ts
var foo = 123;
var bar = 'hey';

bar = foo; // ERROR: –Ω–µ –º–æ–∂–µ –ø—Ä–∏–∑–Ω–∞—á–∏—Ç–∏ number –≤ string
```

–ù–µ–∑–≤–∞–∂–∞—é—á–∏ –Ω–∞ —Ç–µ, —â–æ **error is valid** (—ñ –≤ –±—ñ–ª—å—à–æ—Å—Ç—ñ –≤–∏–ø–∞–¥–∫—ñ–≤ –æ—Ç—Ä–∏–º–∞–Ω–∞ —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—è –±—É–¥–µ –∫—Ä–∞—â–æ—é, –Ω—ñ–∂ —Ç–µ, —â–æ —É—è–≤–ª—è–ª–∏ –æ—Ä–∏–≥—ñ–Ω–∞–ª—å–Ω—ñ –∞–≤—Ç–æ—Ä–∏ —Ä—ñ–∑–Ω–∏—Ö —á–∞—Å—Ç–∏–Ω –±–∞–∑–∏ –∫–æ–¥—É), –≤–∏, –π–º–æ–≤—ñ—Ä–Ω–æ, –∑–æ—Å–µ—Ä–µ–¥–∏—Ç–µ—Å—è –Ω–∞ –Ω–∞–ø–∏—Å–∞–Ω–Ω—ñ –Ω–æ–≤–æ–≥–æ –∫–æ–¥—É –Ω–∞ TypeScript, –ø–æ—Å—Ç—É–ø–æ–≤–æ –æ–Ω–æ–≤–ª—é—é—á–∏ —Å—Ç–∞—Ä—É –∫–æ–¥–æ–≤—É –±–∞–∑—É. –¢—É—Ç –≤–∏ –º–æ–∂–µ—Ç–µ –ø—Ä–∏–¥—É—à–∏—Ç–∏ —Ü—é –ø–æ–º–∏–ª–∫—É –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é —Ç–≤–µ—Ä–¥–∂–µ–Ω–Ω—è —Ç–∏–ø—É, —è–∫ –ø–æ–∫–∞–∑–∞–Ω–æ –Ω–∏–∂—á–µ:

```ts
var foo = 123;
var bar = 'hey';

```

–í —ñ–Ω—à–∏—Ö –º—ñ—Å—Ü—è—Ö –≤–∏ –º–æ–∂–µ—Ç–µ –ø–æ–∑–Ω–∞—á–∏—Ç–∏ —â–æ—Å—å —è–∫ `any`, –Ω–∞–ø—Ä–∏–∫–ª–∞–¥:

```ts

    return 1;
```


> –ü—Ä–∏–º—ñ—Ç–∫–∞: –ø—Ä–∏–¥—É—à–µ–Ω–Ω—è –ø–æ–º–∏–ª–æ–∫ –Ω–µ–±–µ–∑–ø–µ—á–Ω–æ, –∞–ª–µ —Ü–µ –¥–æ–∑–≤–æ–ª—è—î –≤–∞–º –ø–æ–º—ñ—á–∞—Ç–∏ –ø–æ–º–∏–ª–∫–∏ —É –≤–∞—à–æ–º—É *new* –∫–æ–¥—ñ TypeScript. –í–∏ –º–æ–∂–µ—Ç–µ –∑–∞–ª–∏—à–∞—Ç–∏ –∫–æ–º–µ–Ω—Ç–∞—Ä—ñ `// TODO:` –ø—ñ–¥ —á–∞—Å —Ä–æ–±–æ—Ç–∏.**

### Third Party JavaScript
–í–∏ –º–æ–∂–µ—Ç–µ –∑–º—ñ–Ω–∏—Ç–∏ —Å–≤—ñ–π JavaScript –Ω–∞ TypeScript, –∞–ª–µ –≤–∏ –Ω–µ –º–æ–∂–µ—Ç–µ –∑–º—ñ–Ω–∏—Ç–∏ –≤–µ—Å—å —Å–≤—ñ—Ç –Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è TypeScript. –û—Å—å —Ç—É—Ç —ñ –∑‚Äô—è–≤–ª—è—î—Ç—å—Å—è –ø—ñ–¥—Ç—Ä–∏–º–∫–∞ –Ω–∞–≤–∫–æ–ª–∏—à–Ω—å–æ–≥–æ –≤–∏–∑–Ω–∞—á–µ–Ω–Ω—è TypeScript. –ù–∞ –ø–æ—á–∞—Ç–∫—É –º–∏ —Ä–µ–∫–æ–º–µ–Ω–¥—É—î–º–æ –≤–∞–º —Å—Ç–≤–æ—Ä–∏—Ç–∏ `vendor.d.ts` (—Ä–æ–∑—à–∏—Ä–µ–Ω–Ω—è `.d.ts` –≤–∫–∞–∑—É—î –Ω–∞ —Ç–µ, —â–æ —Ü–µ *declaration file*) —ñ –ø–æ—á–∞—Ç–∏ –¥–æ–¥–∞–≤–∞—Ç–∏ –±—Ä—É–¥–Ω—ñ —Ä–µ—á—ñ –¥–æ –Ω—å–æ–≥–æ. –ö—Ä—ñ–º —Ç–æ–≥–æ, –º–æ–∂–Ω–∞ —Å—Ç–≤–æ—Ä–∏—Ç–∏ —Å–ø–µ—Ü—ñ–∞–ª—å–Ω–∏–π —Ñ–∞–π–ª –¥–ª—è –±—ñ–±–ª—ñ–æ—Ç–µ–∫–∏, –Ω–∞–ø—Ä–∏–∫–ª–∞–¥. `jquery.d.ts` –¥–ª—è jquery.

–†–æ–∑–≥–ª—è–Ω–µ–º–æ –≤–∏–ø–∞–¥–æ–∫ `jquery`, –≤–∏ –º–æ–∂–µ—Ç–µ –ª–µ–≥–∫–æ —Å—Ç–≤–æ—Ä–∏—Ç–∏ *trivial* –≤–∏–∑–Ω–∞—á–µ–Ω–Ω—è –¥–ª—è –Ω—å–æ–≥–æ:
